# À lire avant installation

Cette archive contient une extension Brave / Chromium appelée **ChatGPT Voice Flow**.

Elle sert à faciliter l’usage vocal de ChatGPT Web.

Elle peut automatiser :

- l’envoi d’une transcription vocale ;
- la lecture vocale de la réponse ;
- le redémarrage d’une nouvelle transcription ;
- la validation automatique après silence.

## Ce que cette extension n’est pas

Ce n’est pas une extension officielle OpenAI.

Ce n’est pas une application cloud.

Ce n’est pas un service externe.

C’est une extension locale chargée manuellement dans Brave.

## Installation rapide

1. Décompressez l’archive.
2. Ouvrez `brave://extensions/`.
3. Activez le mode développeur.
4. Cliquez sur “Load unpacked”.
5. Sélectionnez le dossier qui contient `manifest.json`.
6. Ouvrez ChatGPT Web.
7. Cliquez sur l’icône de l’extension.
8. Testez doucement, option par option.

## Conseil

Ne mettez pas tout à Yes au premier lancement.

Commencez par tester seulement l’affichage du widget et l’envoi automatique.

Le reste peut être activé ensuite.
