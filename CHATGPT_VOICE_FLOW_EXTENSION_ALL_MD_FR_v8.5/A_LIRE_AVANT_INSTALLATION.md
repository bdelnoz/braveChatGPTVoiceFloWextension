# À lire avant installation

Cette extension automatise ChatGPT Web.

Elle n’est pas officielle.

Elle peut casser si l’interface de ChatGPT change.

## Installation rapide

1. Décompresser le ZIP.
2. Ouvrir `brave://extensions/`.
3. Activer le mode développeur.
4. Cliquer sur “Load unpacked”.
5. Sélectionner le dossier contenant `manifest.json`.
6. Ouvrir ChatGPT Web.
7. Cliquer sur l’icône de l’extension.

## Conseil

Ne pas tout activer au premier lancement.

Tester option par option.

Si quelque chose se comporte bizarrement, désactiver Flow et rafraîchir la page ChatGPT.
