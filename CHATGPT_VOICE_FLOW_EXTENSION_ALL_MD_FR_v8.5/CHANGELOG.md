## v8.5

Point de reprise propre.

Cette version doit servir de base pour la prochaine correction.

## Versions expérimentales à éviter

Les versions suivantes ne doivent pas être utilisées comme base sans vérification :

- v8.7 ;
- v8.8 ;
- v8.9 ;
- v9.0.

Elles ont introduit des régressions importantes dans un autre chat.
